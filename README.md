**Fibonacci Sequence Generator**  
This Python code generates the Fibonacci sequence up to a number entered by the user and saves the result to a text file.  

How to Use  
	1.	Run the script.  
	2.	Enter a number when prompted.  
	3.	The Fibonacci sequence up to that number will be displayed and saved to test1_output.txt in the folder ex_1 on your Desktop.
